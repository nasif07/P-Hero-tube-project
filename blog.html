1. Discuss the scope of var, let, and const?

Ans: var, let, and const are used to declare variables.var declarations are globally scoped. This means that any variable that is declared with var outside a function block is available for use in the whole window. let and const are block-scoped. The scope of let and const is limited to the block in which they are declared.

2. Tell us the use cases of null and undefined

Ans: null is used to define something programmatically empty. Use cases for null include defining something as programmatically empty while use cases for undefined include checking if a variable has been defined or not.

3. What do you mean by REST API?

Ans: API means Application Program Interface. A REST API is an API that conforms to the design principles of the REST, or representational state transfer architectural style.
